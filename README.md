# VeiculoApi
Api de cadastro de veículos

# endpoints
GET /veiculos

POST /veiculos
Exemplo de json de entrada

```json
{
    "ano": 2020,
    "created": "2021-04-20",
    "descricao": "string",
    "marca": "Honda",
    "updated": "2021-04-20",
    "veiculo": "Civic",
    "vendido": true
}
```

# Documentação da Api
Ao rodar o projeto, a documentação será exibida em http://localhost:8080/swagger-ui.html#/
